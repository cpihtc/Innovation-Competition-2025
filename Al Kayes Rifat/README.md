# Amol Nama - Attendance Management System

A simple and efficient attendance management application for tracking student attendance across different subjects and semesters.

## üìã How This App Works

### Overview
Amol Nama is a web-based attendance management system that allows educators to:

1. **User Authentication**
   - Sign up and log in securely using Supabase authentication
   - Each user has their own private dashboard

2. **Subject Management**
   - Create subjects organized by semester
   - View all your subjects in a clean dashboard interface
   - Delete subjects when no longer needed

3. **Student Management**
   - Add students to each subject
   - Maintain a list of enrolled students per subject
   - Remove students as needed

4. **Attendance Tracking**
   - Mark daily attendance for each student (Present/Absent)
   - View attendance records by date
   - Track attendance patterns over time
   - View attendance statistics and reports

### User Flow
1. **Landing Page** ‚Üí Users land on the home page with app information
2. **Authentication** ‚Üí Sign up or log in to access the dashboard
3. **Dashboard** ‚Üí View all subjects, create new ones
4. **Subject Detail** ‚Üí Select a subject to manage students and attendance
5. **Attendance View** ‚Üí Mark and view attendance records
